---

## 🎯 设计理念

### "数字研究员" vs "爬虫"

| 爬虫思维 | 数字研究员思维 |
|---------|--------------|
| 速度优先，批量抓取 | 质量优先，深度理解 |
| 绕过所有限制 | 尊重平台，合规获取 |
| 7x24不间断 | 人类工作时段，有节奏 |
| 纯自动化 | 自动化为主，人工关键验证 |

## 🚀 快速开始

### 1. 环境准备

```bash
cd Project/TalentIntel

# 安装依赖
pip install requests

# 可选: 设置GitHub Token（提高速率限制）
export GITHUB_TOKEN="your_github_token"
```

### 2. 运行完整流程

```bash
# 运行每日完整流程（全自动 + 触发人工验证）
python3 coordinator.py
```

### 3. 单独运行某阶段

```bash
# 仅GitHub挖掘
python3 coordinator.py --phase github

# 仅学术挖掘
python3 coordinator.py --phase academic

# 仅触发验证（基于已有数据）
python3 coordinator.py --phase verify
```

### 4. 运行独立脚本

```bash
# GitHub人才挖掘
python3 scripts/github_talent_miner.py --location "Hong Kong" "California"

# 学术人才挖掘
python3 scripts/academic_talent_miner.py
```

## 👁️ 人工介入指南

### 何时需要人工介入？

系统会自动识别需要人工验证的候选人：
- 匹配分数 **≥ 0.8**
- 每日最多 **3人**

### 验证任务流程

```bash
# 1. 查看待验证任务
cat data/daily/2026-03-20/verify_Dr_Sarah_Chen.json

# 2. 使用agent-browser搜索LinkedIn
agent-browser --headed open "https://www.google.com/search?q=Dr+Sarah+Chen+Stanford+LinkedIn"

# 3. 如遇验证码，手动完成
# 4. 找到LinkedIn档案，复制URL

# 5. 更新验证文件
vim data/daily/2026-03-20/verify_Dr_Sarah_Chen.json
# 添加:
# "linkedin_url": "https://linkedin.com/in/sarah-chen-xxx",
# "status": "human_verified",
# "verification_notes": "已验证，现任Stanford博士后"
```

### 人工介入频率

| 场景 | 频率 | 时长 |
|-----|------|------|
| 日常运行 | 每日 | 0分钟（全自动） |
| 高分验证 | 每日 | 10-30分钟（最多3人） |
| 数据审核 | 每周 | 30分钟 |
